# Supply — Demo (عبدو البحراوي)

نسخة تجريبية بسيطة (Frontend-only demo) تعمل بدون سيرفر.  
طريقة التشغيل:
- افتح الملفات محليًا (انقر مرتين على index.html) أو ارفعها لمستودع GitHub ثم فعّل GitHub Pages (Settings → Pages).
- لإرسال إثباتات الدفع: افتح "إضافة أموال" واملأ النموذج — تظهر الطلبات في "لوحة الأدمن".
- هذه نسخة DEMO تستخدم localStorage فقط ولا تخزن بيانات على سيرفر.
